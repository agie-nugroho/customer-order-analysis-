# 📊 Customer Order Data Analysis

## 📌 Overview

This project demonstrates a basic Exploratory Data Analysis (EDA) workflow using Python.

The dataset contains customer purchasing information such as:

- Order Amount
- Number of Items
- Customer Age
- Discount
- Delivery Time

The project covers data exploration, descriptive statistics, visualization, and correlation analysis using Pandas, Matplotlib, and Seaborn.

---

## 📂 Dataset

| Column | Description |
|---------|-------------|
| Order_Amount | Total order amount |
| Num_Items | Number of purchased items |
| Customer_Age | Customer age |
| Discount | Discount percentage |
| Delivery_Time | Delivery time (minutes) |

Total Records : **10**
